# nix-config

These are my personal Linux dotfiles that I've adapted from the amazing [CelestialCrafter's dotfiles](https://github.com/CelestialCrafter/dotfiles). I've made them into Nix configurations for easier management and customization.

They don't differ much from the original for now, but my goal is to create a minimal setup using RiverWM.
